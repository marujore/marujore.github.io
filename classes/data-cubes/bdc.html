Brazil Data Cube¶

Brazil Data Cube (BDC) project started on 2019 January by INPE (Brazil National Institute for Space Research), with the objective of creating multidimensional medium spatial resolution analysis ready images for the entire Brazilian territory and generate land use and land cover information through machine learning and satellite time series.

Brazil Data Cube is a subproject of the Environmental Monitoring of Brazilian Biomes, funded by ` Amazon Fund <http://www.fundoamazonia.gov.br/pt/projeto/Monitoramento-Ambiental-dos-Biomas-Brasileiros/>`_, through a finantial colaboration with BNDES and FUNCATE nº 17.2.0536.1.

All codes are open source, and can be found in the project official GitHub repository, which includes acquisition and data processing, Data Cube generation, Data Base Model frameworks, web portal page, services as WTSS, and others.

Many of those developped using Python language.

BDC process and makes available images from different sensors, providing several ready to use products, also called Analysys Ready Data (ARD), so researchers can focus on analysis instead of processing. All project products can be seen on https://brazil-data-cube.github.io/products.html. These images processed by BDC are free and can be acquired on the BDC Web Portal, which is done through SpatioTemporal Asset Catalog (STAC).
